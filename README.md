# 🔐 Login & Register System (Intentionally Vulnerable)

⚠️ Warning: This application is intentionally vulnerable and was created for educational and security learning purposes only.

This project contains common web application vulnerabilities to practice:
- Web penetration testing
- Secure coding concepts
- Vulnerability discovery
- Exploitation in controlled environments

## Vulnerabilities Included

### 1. Cross-Site Scripting (XSS)
Input fields may allow malicious scripts to execute in the browser.

### 2. SQL Injection (SQLi)
Some database queries may be vulnerable to SQL injection attacks.

### 3. Broken Access Control
Unauthorized users may access functionality or data that should be restricted.

### 4. Authentication Vulnerabilities
Weak authentication and session handling mechanisms may exist.

### 5. Information Disclosure
Sensitive information may be exposed unintentionally.

## Educational Purpose

This project exists only for:
- Learning web security
- Practicing vulnerability assessment
- Understanding secure development

Do NOT deploy this application publicly.
